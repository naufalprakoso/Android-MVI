# Android-MVI
Superhero and Villain app using MVI Pattern on Android 🦸🏻‍♂️

The app uses the following libraries / topics:
- Kotlin
- Coroutines
- Retrofit2
- Glide
- ViewModels
- Repository pattern
- NetworkBoundResource.

API:
Characters data from https://akabab.github.io/superhero-api/api/.
